Por que ser skatista ?

Postado por Bryan Santos

Os motivos e as sensações variam de skatista para skatista. Para alguns o skate é pura diversão, um momento de estar entre amigos e conhecer novas pessoas. Para outros é testar os próprios “limites”. Para muitos é a oportunidade de conhecer lugares e culturas diferentes. Para outros é uma forma de ganhar dinheiro. Para alguns é praticar uma atividade física. Para muitos é manter a mente e o corpo em equilíbrio. Para algumas pessoas é sentir a felicidade e alívio por acertar uma manobra que passou horas tentando. Para outros é terminar a sessão feliz por estar curtindo o momento, independente de ter acertado ou não manobras. Para alguns andar de skate é sentir o vento no rosto. Para outros é se sentir livre. Para alguns é aprender mais a cada dia. Para outros é dividir com o próximo tudo o que aprendeu com o skate.
